# LALG Scanner

Este projeto implementa um analisador léxico para a linguagem LALG. Ele consiste em uma interface web onde o usuário pode inserir código LALG e visualizar a lista de tokens gerada pelo scanner.

## Tecnologias Utilizadas

*   **Backend:** Python (FastAPI)
*   **Frontend:** HTML, CSS, JavaScript
